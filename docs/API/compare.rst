Compare Module
=============

The compare module provides functions for comparing multiple genome annotations.

Key Functions
------------

compare
~~~~~~

.. autofunction:: funannotate2.compare.compare

This is the main function for comparing multiple genome annotations. It takes multiple annotation directories as input and produces various comparison files as output.

check_inputs
~~~~~~~~~~~

.. autofunction:: funannotate2.compare.check_inputs

Checks that the input files exist and are valid.

find_input_files
~~~~~~~~~~~~~

.. autofunction:: funannotate2.compare.find_input_files

Finds input files in the specified directories.

parse_annotations
~~~~~~~~~~~~~~

.. autofunction:: funannotate2.compare.parse_annotations

compare_annotations
~~~~~~~~~~~~~~~

.. autofunction:: funannotate2.compare.compare_annotations

Compares annotations from multiple sources.

generate_reports
~~~~~~~~~~~~~

.. autofunction:: funannotate2.compare.generate_reports
